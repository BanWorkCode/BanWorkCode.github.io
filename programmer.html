Автономный программатор

Основные Функции

- Загрузка прошивки в программируемый микроконтроллер STM32 из HEX-фала через UART.

- Получение файла прошивки через MicroSD карту или с FTP сервера через Wi-Fi.

- Обновление собственной прошивки по воздуху, с помощью встроенного загрузчика.
